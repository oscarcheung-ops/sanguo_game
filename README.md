# 🎮 三國戰記 - 手機版

一款基於三國題材的即時戰略遊戲，專為手機瀏覽器優化。

## 🎯 版本說明

### 🎮 完整版 (推薦) - `index-full.html`
- 6位英雄（關羽、張飛、趙雲、馬超、黃忠、黃月英）
- 完整英雄專精系統
- 星級升級和屬性加成
- 複雜戰鬥系統和技能
- 主菜單、隊伍編成、抽卡系統
- **手機最佳體驗**

### 🚀 簡化版 - `index.html`
- 快速開始遊戲
- 三兵種系統
- 基本升級機制
- 輕量級設計

## 🎮 遊戲特色

- 📱 **完美支援手機** - 觸控優化，適配各種螢幕尺寸
- ⚔️ **三兵種系統** - 槍兵、騎兵、弓兵，兵種相剋
- 🎭 **6位英雄** - 每個英雄有獨特的專精能力
- 🏰 **城堡攻防** - 保護己方城堡，摧毀敵方城堡
- 💰 **策略升級** - 累積金幣，升級部隊屬性和星級
- 🤖 **自動模式** - 支援自動戰鬥，輕鬆掛機
- ⚡ **速度調整** - 支援 1.0x, 1.5x, 2.0x, 3.0x 遊戲速度
- 🌊 **多章節挑戰** - 關卡越高，敵人越強

## 🎮 遊戲玩法

### 英雄介紹

| 英雄 | 兵種 | 特性 | 專精能力 |
|------|------|------|--------|
| 🔱 關羽 | 槍兵 | 攻擊型 | 技能冷卻-20% |
| 🔱 張飛 | 槍兵 | 坦克型 | 戰鬥中回復10% HP/s |
| 🐎 趙雲 | 騎兵 | 輸出型 | 攻擊力+15% |
| 🐎 馬超 | 騎兵 | 速度型 | 移動速度+25% |
| 🏹 黃忠 | 弓兵 | 遠程型 | 暴擊率+30% |
| 🏹 黃月英 | 弓兵 | 法師型 | 技能傷害+30% |

### 兵種相剋

```
槍兵 → 克制 → 騎兵
  ↑             ↓
  克           克制
  制             ↓
  ↑           弓兵
```

- 槍兵：100 HP，20 ATK，中等速度 - **克制騎兵**
- 騎兵：80 HP，25 ATK，快速 - **克制弓兵**
- 弓兵：70 HP，28 ATK，遠程 - **克制槍兵**

### 兵種技能

- **槍兵 - 貫穿突刺**：對前方敵人造成150%傷害+25%概率眩暈
- **騎兵 - 衝鋒突擊**：衝向敵人造成180%傷害並減速50%，自身恢復25% HP
- **弓兵 - 連射覆蓋**：向範圍內射出3支箭，每支造成120%傷害，目標減速

### 戰鬥系統

1. **隊伍編成** - 選擇最多3個英雄組成隊伍
2. **波次戰鬥** - 逐波擊敗敵人
3. **自動戰鬥** - 開啟自動模式自動進行
4. **獲得獎勵** - 擊敗敵人獲得金幣

### 進度系統

- 📊 **等級系統** - 等級越高，敵人越強，但屬性加成也越多
- ⭐ **星級系統** - 升級英雄星級以獲得屬性加成
- 💎 **抽卡系統** - 使用鑽石抽卡獲取英雄

## 🚀 如何遊玩

### 在線遊玩
**完整版**：https://oscarcheung-ops.github.io/sanguo_game/index-full.html  
**簡化版**：https://oscarcheung-ops.github.io/sanguo_game/

### 本地遊玩
1. 下載或克隆此倉庫
2. 用瀏覽器打開 `index-full.html`（完整版）或 `index.html`（簡化版）
3. 開始遊戲！

## 📱 手機訪問

1. 在手機瀏覽器打開遊戲連結
2. 建議添加到主畫面，獲得類似 APP 的體驗

### iOS (Safari)
- 點擊分享按鈕 → 加入主畫面

### Android (Chrome)
- 點擊選單 → 加到主畫面
