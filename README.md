# 🎮 三國戰記 - 手機版

一款基於三國題材的即時戰略遊戲，專為手機瀏覽器優化。

## 🎯 遊戲特色

- 📱 **完美支援手機** - 觸控優化，適配各種螢幕尺寸
- ⚔️ **三兵種系統** - 槍兵、騎兵、弓兵，兵種相剋
- 🏰 **城堡攻防** - 保護己方城堡，摧毀敵方城堡
- 💰 **策略升級** - 累積金幣，升級部隊屬性
- 🤖 **自動模式** - 支援自動戰鬥，輕鬆掛機
- 🌊 **無盡關卡** - 關卡越高，挑戰越大

## 🎮 遊戲玩法

### 基本操作
- 點擊下方按鈕召喚不同兵種
- 槍兵 (💰50) - 生命高，克制騎兵
- 騎兵 (💰60) - 速度快，克制弓兵  
- 弓兵 (💰55) - 射程遠，克制槍兵

### 兵種相剋
```
槍兵 → 克制 → 騎兵
  ↑             ↓
  克           克制
  制             ↓
  ↑           弓兵
```

### 策略升級
使用金幣 (💰100) 升級：
- 🗡️ 攻擊力 +20%
- ❤️ 生命值 +20%
- 🏃 移動速度 +20%

### 獲勝條件
- 摧毀敵方城堡即可過關
- 每關獲得 +50 金幣獎勵
- 關卡越高，敵人越強

## 🚀 如何遊玩

### 在線遊玩
直接訪問：[https://你的用戶名.github.io/sanguo-game/](https://你的用戶名.github.io/sanguo-game/)

### 本地遊玩
1. 下載或克隆此倉庫
2. 用瀏覽器打開 `index.html`
3. 開始遊戲！

## 📱 手機訪問

1. 在手機瀏覽器打開遊戲連結
2. 建議添加到主畫面，獲得類似 APP 的體驗
3. 支援全螢幕模式

### iOS (Safari)
- 點擊分享按鈕 → 加入主畫面

### Android (Chrome)
- 點擊選單 → 加到主畫面
